## üìñ Introduction

**BookLibrary** is a comprehensive library management and delivery system designed to bridge the gap between physical libraries and readers. It revolutionizes the traditional library experience by allowing users to borrow books online and have them delivered directly to their doorstep, eliminating the need for physical visits.

The platform features a robust **multi-role ecosystem** (User, Librarian, Admin) where librarians can manage their catalog, users can track orders in real-time, and admins maintain system integrity.

## ‚ú® Key Features

### üë§ For Users

- **Browse & Search:** Explore a curated collection of books with advanced search and sorting by price.
- **Book Details:** View detailed information, ratings, and reviews before borrowing.
- **Wishlist:** Save favorite books for later.
- **Order Management:** Seamless ordering process with **Real-time status tracking** (Pending ‚Üí Shipped ‚Üí Delivered).
- **Payment Integration:** Secure "Pay Now" functionality for pending orders via **Stripe**.
- **Profile Management:** Update personal details and profile picture.

### üìö For Librarians

- **Catalog Management:** Add new books, edit existing details, and manage publication status (Publish/Unpublish).
- **Order Fulfillment:** View incoming orders and update statuses (Shipped/Delivered).
- **My Books:** A dedicated dashboard to track books added by the specific librarian.

### üõ°Ô∏è For Admins

- **User Management:** Promote users to Librarian or Admin roles.
- **System Oversight:** Manage all books across the platform, including the ability to delete or unpublish inappropriate content.
- **Stats Dashboard:** Visualize platform data (Total Users, Books, Orders, Revenue).
